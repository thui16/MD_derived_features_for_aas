Folder or file; description
============================
/numpys; The enclosed files are the position-aware sidechain (PASC) features for the amino acids included in our manuscript, in numpy format. 
encoding_with_PASC.py; An example python script that will take in a text file (containing a peptide sequence(s)) and output their PASC feature encoded vector in csv format. Example command: 'python encoding_with_PASC.py --f example_sequences.txt' 

example_sequences.txt; An example text file that will be input into encoding_with_PASC.py

PASC_encoded_example_sequences.csv; An example output file from encoding_with_PASC.py, given the example_sequences.txt as input.
